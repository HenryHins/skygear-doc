# Using npm

``` bash
npm install skygear --save
```

# Install the latest

``` bash
git submodule git@github.com:oursky/skygear.git 
cd skygear
npm install
cd ..
npm install skygear/
```

# Includ in your website

``` html
<script src="//cdn.skygear.io/skygear.js"></script>
```
