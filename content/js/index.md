# Using npm

``` bash
npm install jsoud --save
```

# Install the latest

``` bash
git submodule git@github.com:oursky/jsourd.git 
cd jsourd
npm install
cd ..
npm install jsoud/
```

# Includ in your website

``` html
<script src="//cdn.jsourd.io/jsourd.js"></script>
```
